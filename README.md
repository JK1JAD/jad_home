## 🛠️ 今後の構成予定
- `/images/`  
  マニュアルに使うスクリーンショット置き場。  
  ※でもつい「家族写真」混ぜてしまう危険あり。  

- `/docs/`  
  PDF版マニュアルや補足資料。  
  ※「重すぎて開かない」って怒られる可能性あり。  

- `index.html`  
  GitHub Pages の公開用トップページ。  
  ※何も置かないと「真っ白ページ」になって泣く。

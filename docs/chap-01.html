第1章 概要

ここではマニュアル全体像と用語をざっくり把握します。PDFは保存用、docsは即読用。三兄弟（PDF・docs・images）が連携して「読める・配れる・直せる」を実現します。

JADワンポイント： 迷子になったら「トップ → docs → 第1章」にQSY。帰巣本能、発動！

基本ディレクトリ

/ (ルート) ├─ index.html ← トップ ├─ pdfs/ ← PDFアーカイブ ├─ docs/ ← いま読んでるオンライン本文 └─ images/ ← スクショ置き場（本文・PDFの共通素材）

画像の参照例

docsからimagesを参照する場合、相対パスは ../images/xxx.png を使います。

執筆ポリシー（ふじさわ節 3箇条）

  1. 最初の1画面で次に何をするか分かる。
  2. スクショは赤矢印で「押す場所」を示す。
  3. ボケは控えめに、ツッコミは的確に（※盛りたい時は最後にまとめて）。

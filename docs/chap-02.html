第2章 構成

ここではサイト全体の配置ルールを決めます。三兄弟（PDF・docs・images）に加え、将来の「samples/」「assets/」なども拡張しやすくします。

推奨パスと命名

/pdfs/manual_vX.Y.Z.pdf /docs/chap-XX.html /images/<用途>/<番号>_説明.png

リンクの基本

  • PDFアーカイブ → ../pdfs/index.html
  • オンライン本文トップ → ./index.html
  • 画像参照（docs→images）→ ../images/…
